# Лабораторная работа №6 (Java)

Тема: Методы, параметры методов и рекурсивные функции.

## Что реализовано
1. `rectangleArea(double width, double height)` — площадь прямоугольника  
2. `isEven(int number)` — проверка чётности  
3. Перегрузка `max(int a, int b)` и `max(double a, double b)`  
4. `factorial(int n)` — рекурсивный факториал + проверка n < 0  
5. `power(int base, int exponent)` — рекурсивная степень (exponent >= 0)
